# Hand Gesture Recognition and Virtual Air Writing

## Project Overview
This project uses OpenCV and MediaPipe to detect hand gestures and perform virtual air writing using a webcam.

## Features
- Hand Detection
- Finger Counting
- Gesture Recognition
- Air Writing

## Technologies Used
- Python
- OpenCV
- MediaPipe
- NumPy

## How to Run
1. Install requirements
2. Run:
   python main.py
